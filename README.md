# Eccentric Aspects

This is a memorial archive of the personal blog of **Nick Wormley**
("cynick"), a railroad historian, photographer, and software engineer
from Lowertown, Saint Paul. The original site at `eccentricaspects.com`
was built with Hakyll and went dark when the hosting lapsed. It has been
restored from archived copies and is preserved here as a permanent
memorial. No new content will be published.
